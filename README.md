# 📄 Currículo - João Gabriel Coelho

Este é um projeto de currículo digital responsivo, estruturado com **HTML5** e estilizado com **CSS3** utilizando as técnicas de **CSS Grid Layout** e **Flexbox**.

## 🚀 Tecnologias Utilizadas

* **HTML5**: Estruturação semântica do conteúdo.
* **CSS3**: Design responsivo, Grid Area para o layout principal e Flexbox para os cartões de habilidades.
* **Google Fonts**: Tipografia personalizada para uma leitura clara.

## 🛠️ Funcionalidades

* **Layout Grid**: Organização dividida em áreas específicas: cabeçalho, navegação lateral, conteúdo principal, barra lateral e rodapé.
* **Navegação Rápida**: Menu com links de âncora para saltar entre as secções de Resumo, Formação, Habilidades e Experiência.
* **Cartões Interativos**: Exibição de competências e formação académica com efeitos visual ao passar o rato (*hover*).
* **Badges de Status**: Etiquetas visuais para identificar rapidamente formações em andamento ou concluídas.
